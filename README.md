## Assignments Overview

### [Assignment 1 - Gender Cleaning](./Assignment%201%20-%20Gender%20Cleaning/)
Data cleaning of the Gender column - standardized values from 6 different representations to binary encoding.

### [Assignment 2 - Pulse Cleaning](./Assignment%202%20-%20Pulse%20Cleaning/)
Cleaned pulse data by handling outliers, non-numeric values, and missing data.

### [Assignment 3 - Data Visualization](./Assignment%203%20-%20Data%20Visualization/)
Created clinical visualizations including gender distribution, pulse histograms, and clinical category charts.

### [Assignment 4 - Triage Pseudocode](./Assignment%204%20-%20Triage%20Pseudocode/)
Designed a digital triaging algorithm using blood pressure and SpO2 measurements.

### [Assignment 5 - Unconsidered Metrics](./Assignment%205%20-%20Unconsidered%20Metrics/)
Analysis of vital signs not covered in primary assignments.

### [Assignment 6 - Career Challenge](./Assignment%206%20-%20Career%20Challenge/)
Reflection on learning journey and career goals in clinical AI.

### [Assignment 7 - Final Submission](./Assignment%207%20-%20Final%20Submission/)
Summary of all completed work and skills developed.
